# Changelog

## 0.1.0

- Chrome Manifest V3 side panel extension.
- YouTube transcript capture with current timestamp awareness.
- Best-effort Netflix, Udemy, Coursera, Bilibili, and active-tab video context capture.
- Chrome built-in AI, OpenRouter, Gemini, Groq, OpenAI, and Ollama providers.
- Direct transcript answers for short factual questions.
- Prompt budgeting for long videos.
- Enter-to-send chat composer.
- Voice questions with friendlier microphone permission errors.
- Optional spoken answers with browser text-to-speech.
- Light, dark, and system theme modes.
- Explicit answer-language control for prompts and direct transcript answers.
- Local validation script and GitHub Actions workflow.
- Optional Chrome for Testing/Chromium smoke test.
